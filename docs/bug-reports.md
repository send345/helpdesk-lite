# Баг-репорты (учебные)

## Баг №1
- **Описание:** При создании заявки можно указать пустой title
- **Шаги:** POST /tickets с title = ""
- **Ожидаемый результат:** Ошибка 400
- **Фактический результат:** Заявка создаётся
- **Статус:** Исправлено добавлением валидации

## Баг №2
- **Описание:** Можно добавить комментарий к несуществующей заявке
- **Шаги:** POST /tickets/999/comments
- **Ожидаемый результат:** Ошибка 404
- **Фактический результат:** Комментарий добавлялся
- **Статус:** Исправлено проверкой существования заявки

## Баг №3
- **Описание:** Можно добавить пустой комментарий
- **Шаги:** POST /tickets/1/comments с text = ""
- **Ожидаемый результат:** Ошибка 400
- **Фактический результат:** Пустой комментарий сохранялся
- **Статус:** Исправлено добавлением проверки на пустоту
