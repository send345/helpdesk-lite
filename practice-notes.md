# Мои заметки по проекту HelpDesk Lite

## Спринт 1

### Что я узнал:

**Git:**
- Git — это система контроля версий
- `git init` — создать репозиторий
- `git add` — добавить файлы
- `git commit` — сохранить изменения
- `git tag` — создать метку

**FastAPI:**
- `@app.get()` — создать эндпоинт для GET-запросов
- Uvicorn — сервер для запуска FastAPI

**HTTP:**
- GET — метод для получения данных
- Параметры в URL: `/hello?name=Ivan`
- localhost:8000 — адрес сервера на моём компьютере
- Статус 200 — успешный ответ

**Терминал (Linux/Windows cmd):**
- `cd` — перейти в папку
- `dir` / `ls` — посмотреть файлы
- `Ctrl+C` — остановить работающую программу
